Projet 5 : NLP : création de tags pour les questions du site stackoverflow

Les bases de données et fichiers annexes ne sont pas intégrées aux scripts de codes ( poids fichiers)


script_creation_word2vec_tsne: création du modèle Word2Vec adapté au projet et visualisation t-SNE
script_lda: création de sujets par la méthode LDA
script_nmf : création de sujets avec NMF
script_tfidf : création d'une matrice des poids des ngrams puis utilisation de classifieur pour trouver les tags les plus adaptés aux questions
script_word2vec_clustering : application d'un partitionnement sur le modèle word2vec
word2vec_classif : utilisation du modèle Word2Vec dans la classification ( le modèle manque de mots pour être véritablement efficace )
script_flask : création d'une API qui, pour une question, retourne les tags les plus appropriés.
